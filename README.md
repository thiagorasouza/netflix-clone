# :tv: Netflix Clone

A pretty accurate Netflix's frontpage clone, implementing most of the details and design using only **Vanilla Javascript** and **Modern CSS** features.

## :fire: Main Features

- Vanilla Javascript
- SASS
- CSS Transitions & Transformations
- Dynamic Video Preview
- HTML 5
